# Orbit Fighter Prototype 

Protótipo experimental de combate 2D desenvolvido no **GameMaker Studio 2**. Este projeto foca no estudo de física vetorial, detecção de colisão circular e sistemas de entidades acopladas.

## 🚀 Objetivos Técnicos
- **Física de Inércia:** Movimentação baseada em aceleração e fricção para simular o vácuo.
- **Parentamento Dinâmico:** Armas que orbitam e rotacionam em relação ao centro do jogador.
- **Clean Code em GML:** Organização de scripts e eventos seguindo padrões de nomenclatura da comunidade.

## 🛠️ Tecnologias
- **Engine:** GameMaker Studio 2 (GML)
- **Versionamento:** Git

## 🎮 Como testar
1. Clone o repositório.
2. Abra o arquivo `.yyp` no GMS2.
3. Pressione `F5` para rodar.
